# Rock-Paper-Scissors

A web-based Rock-Paper-Scissors game built as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Live Demo

[Play the game here](https://thechosen-1.github.io/Rock-Paper-Scissors/)

## About

This project is part of my learning journey through The Odin Project's Foundations course. It demonstrates fundamental JavaScript concepts including DOM manipulation, event handling, and game logic.

## Features

- Play against the computer
- First to 5 wins takes the game
- Score tracking for both player and computer
- Clean, simple UI with interactive buttons

## Technologies Used

- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript

## What I Learned

- DOM manipulation and event listeners
- Game state management with JavaScript
- Conditional logic for game rules
- Dynamic content updates without page reload
